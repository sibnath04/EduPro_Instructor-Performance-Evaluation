# Instructor Performance and Course Quality Evaluation on EduPro

## Project Overview

This project analyzes instructor performance and course quality using data analytics techniques. It helps identify high-performing instructors, evaluate course quality, analyze the impact of teaching experience, and support data-driven decision-making.

---

## Objectives

- Evaluate instructor performance.
- Analyze course quality across categories.
- Study the relationship between teaching experience and ratings.
- Identify top-performing instructors.
- Analyze enrollment trends.
- Build an interactive Streamlit dashboard.

---

## Dataset

The project uses four datasets:

- Teachers
- Courses
- Transactions
- Users

The datasets are merged using:

- TeacherID
- CourseID
- UserID
